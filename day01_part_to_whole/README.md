# Day 1: Part-to-Whole - LLM Usage by Developers (2025)

## Challenge Details
- **Day**: 1
- **Category**: Comparisons
- **Theme**: Part-to-Whole
- **Date**: April 1, 2026

## Visualization Description
This visualization shows the distribution of Large Language Model (LLM) usage among developers in 2025, illustrating which AI models developers prefer and how they contribute to the total AI tool adoption landscape.

## Data Source
**Source**: Stack Overflow Developer Survey 2025
**URL**: https://survey.stackoverflow.co/2025/
**Survey Period**: 2025
**Respondents**: 49,000+ developers from 177 countries
**Specific Section**: AI & LLM Usage

### Key Statistics Referenced:
- OpenAI GPT Models: 81% of developers
- Claude Sonnet: 43% of professional developers
- Gemini Flash: 35% of developers
- Additional models: Llama, Mistral, and others

**Press Release**: https://stackoverflow.co/company/press/archive/stack-overflow-2025-developer-survey/

## Tools Used
- **Language**: Python 3.x
- **Libraries**: matplotlib, pandas, seaborn

## Files
- `data/llm_usage_2025.csv` - Dataset with LLM usage percentages
- `scripts/part_to_whole_viz.py` - Python script for visualization
- `output/` - Generated visualization images

## How to Run
```bash
cd day01_part_to_whole
python scripts/part_to_whole_viz.py
```

## Notes
The percentages may sum to more than 100% as developers often use multiple LLM models in their workflow.
